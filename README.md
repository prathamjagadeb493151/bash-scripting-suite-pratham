#  System Monitor Tool using C++
**Capstone Project — Wipro | ITER College**  
**Author:** Pratham Jagadeb  

---

##  Project Overview
This project is a Linux-based **System Monitor Tool** built in **C++**, which displays real-time CPU and memory usage, uptime, and active process information using the **/proc filesystem** and the **ncurses** library.

---

##  Objective
To develop a terminal-based system monitor that retrieves and displays key performance metrics such as CPU %, memory utilization, process IDs, and system uptime.

---

##  Technologies Used
| Component | Description |
|------------|-------------|
| **Language** | C++17 |
| **Platform** | Linux (Ubuntu / WSL) |
| **Library** | ncurses |
| **System Interface** | /proc filesystem |
